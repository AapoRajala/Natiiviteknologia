# VIIKKO 1

## Domain + Kotlin-harjoitukset + ensimmäinen Compose-näkymä

### MOCKDATA

Mockdata.kt sisältää tietoa jotka on annettu jo ennen itse koodin ajamista.
Tässä projektissa mockdataan oli alustettu tiedot: id, title, description, priotity, date ja done

### KotlinFunction

KotlinFunctions tiedostossa on funktiot jota ajetaan "MainActivity" tiedosta käsin.
olen tehnyt kyseiseen tiedostoon funktiot jolla pystyy päivittämään booleanin Falsesta -> Trueksi tai toisinpäin, sekä
Päivämäärän filteröinti on lisätty, myöskin tein vielä päivämäärien filtteröinin ja taskien lisäämisen.

### MainActivity

MainActivityssä ajetaan koodin fronttipuoli, mitä itse käyttäjä näkee.
Tein yhteydet KotlinFunctionsiin ja lisäsin napit jolla pystyi lisäämään joko taskeja, muokkaamaan niitä True/ False tai muuten vain sorttaamaan.
